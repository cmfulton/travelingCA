Running mcmc.py (MCMC for Traveling Salesman Problem and MCMC with Simulated Annealing)

1. This uses cmf_cities.csv so make sure its there
2. Run mcmc.py
3. Output will be printed to the console and also html maps will be created under mcmc_maps_results and
    mcmc_sa_maps_results
4. The shortest path html maps will be written to the same directory as the mcmc.py file

Running market_basket_analysis.py

1. This uses result.csv (needs to be in same directory as market_basket_analysis.py)
2. Output will be written to the console. This takes a while to run
